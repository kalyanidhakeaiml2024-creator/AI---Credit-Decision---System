## Project Description

An AI-powered Loan Approval Prediction System developed using Machine Learning and Flask. The system predicts whether a loan application is likely to be approved based on applicant details. It also generates a professional CAM (Credit Appraisal Memorandum) report.

---

## Features

- AI-based Loan Approval Prediction
- Applicant Details Form
- Risk Level Analysis
- CAM Report Generation
- Downloadable PDF Report
- Responsive Web Interface
- Flask Backend
- Machine Learning Model

---

## Technologies Used

- Python
- Flask
- HTML
- CSS

- Scikit-Learn
